1. Editar o arquivo 'grafo.txt' de acordo com as seguintes diretrizes:

* A primeira linha fornece o número de vértices do grafo;
* A segunda linha consiste em um único número explicitando de qual vértice deverá ser impresso os menores caminhos;
* As linhas seguintes são compostas de dois inteiros e um valor real que representam uma aresta e seu peso. Por exemplo a tripla {i j s} diz que o
grafo possui uma aresta de i para j com peso s;

2. Executar o arquivo main.py contido nesta pasta a partir de um interpretador python.

OBS: O contexto de execução deve partir de dentro desta pasta "Floyd-Warshall", do contrário o algoritmo não encontrará o arquivo grafo.txt
